# RC-TO-SERIAL
 RC Command Translator
 Send MABC Commands to your Droid via RC Transmitter
 
 
## PINS for use with Arduino Pro Micro
* 2 - 12  RC Input
* 0 TX for 9600 Baud Output
* PIN  2-3-4-5-6-7-8-9

* UP      Sound +        SHAKE      CantinaSE07    LeiaSE08    SmirkSE03        Open All     Arms OPA     Flutter       Sound Next$2      SHAKE
*    ||---1--CH8---||---2-CH11---||---2-CH12---||---3-CH13--||---4-CH14--|    |---5-CH10--||---6-CH9---||---7-CH9----||-----8-CH10----||---9-CH11----|  
* Down    Sound -         232        Manamana     FaintSE06    Wave2SE04        AwakeSE14     SPRAY       ResetSE00        Mode           move

## CH SET
* CH8   VOL -  / VOL +
          
* CH11  SHAKE / 232
  
* CH12  Cantina / Manama
  
* CH13  Leia / Faint
    
* CH14  Smirk / Wave 

* CH10   OP ALL / Awake
    
* CH9   ARM  / Spray
     
* CH6   Flutter / Reset
     
* CH5   Next Sound / Mode

* CH7   Source RC or Stick
