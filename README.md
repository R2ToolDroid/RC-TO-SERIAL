# RC-TO-SERIAL
 RC Command Translator
 Send MABC Commands to your Droid via RC Transmitter
 
 
## PINS for use with Arduino Pro Micro
* 2 - 12  RC Input
* 0 TX for 9600 Baud Output
* PIN  2-3-4-5-6-7-8-9


## CH SET
* CH8   VOL -  / VOL +
          
* CH11  SHAKE / 232
  
* CH12  Cantina / Manama
  
* CH13  Leia / Faint
    
* CH14  Smirk / Wave 

* CH10   OP ALL / Awake
    
* CH9   ARM  / Spray
     
* CH6   Flutter / Reset
     
* CH5   Next Sound / Mode

* CH7   Source RC or Stick
